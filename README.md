## О проекте

Исследование рынка авиаперевозок Индии на основе **300 000 реальных бронирований**.  
Проект выполнен как тестовое задание для Aviasales — от SQL-запросов до бизнес-рекомендаций.

### Данные

- **300 153** бронирования
- **6** авиакомпаний: Vistara, Air_India, Indigo, SpiceJet, GO_FIRST, AirAsia
- **6** городов: Дели, Мумбаи, Бангалор, Ченнаи, Калькутта, Хайдарабад
- Признаки: цена, класс, дни до вылета, пересадки, время вылета, длительность

### Инструменты

`SQL` `Python` `pandas` `numpy` `matplotlib` `seaborn`

## Что внутри

### 1. SQL — JOIN, CTE, оконные функции

5 сложных запросов на `pandasql`:

| Запрос | Что сделано |
|--------|------------|
| Ранг авиакомпаний | `RANK() OVER (PARTITION BY class)` — кто лидер по цене в каждом классе |
| Кумулятивная выручка | `SUM() OVER (ORDER BY days_left)` — накопленная выручка по дням до вылета |
| Конверсия по направлениям | Два CTE + `JOIN` — доля каждого класса на маршруте |
| Скользящее среднее | `ROWS BETWEEN 3 PRECEDING AND 3 FOLLOWING` — сглаживание за 7 дней |
| Топ-3 по направлениям | `ROW_NUMBER() OVER (PARTITION BY route)` — лидеры на каждом маршруте |

### 2. Статистический анализ

- **Меры центральной тенденции**: mean, median, mode
- **Меры вариации**: стандартное отклонение, IQR, коэффициент вариации
- **Процентили**: P1–P99, анализ разброса цен
- **Правило Парето**: топ-10% билетов приносят 32% выручки
- **95% доверительный интервал** для средней цены билета

### 3. BI-дашборд

Визуализация в стиле Aviasales (оранжевый + синий):

- **KPI-карточки**: всего бронирований, выручка, средний чек, направлений, авиакомпаний
- **Топ-10 направлений**, выручка по авиакомпаниям, средний чек по классам
- **Распределение цен** со средним и медианой
- **Динамика цены** по дням до вылета
- **Разброс цен** по авиакомпаниям (boxplot)
- **RFM-сегментация**: VIP, лояльные, перспективные, спящие, ушедшие

### 4. Бизнес-выводы

7 конкретных рекомендаций для продукта Aviasales:

| Инсайт | Рекомендация |
|--------|-------------|
| 69% ищут прямые рейсы | Фильтр «Без пересадок» включить по умолчанию |
| 10% билетов = 32% выручки | Премиум-подписка для бизнес-класса |
| Два типа пользователей (26 дн / 7 дн) | Разный UX для планировщиков и last-minute |
| Vistara + Air_India = 75% рынка | Эксклюзивное партнёрство с лидерами |
| Цена не падает перед вылетом | Показать график — пользователь перестанет ждать |
| 18% пользователей = основной доход | Персонализация для VIP и лояльных |
| Бизнес-класс в 8 раз дороже эконома | Разный сервис для разных сегментов |
