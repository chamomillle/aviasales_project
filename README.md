## Структура проекта

### Часть 1. SQL — JOIN, CTE, оконные функции

Показано уверенное владение SQL. Все запросы написаны через `pandasql` и используют реальные данные.

#### Запрос 1: Ранг авиакомпаний по средней цене внутри класса
```sql
SELECT 
    airline, class,
    ROUND(AVG(price), 0) AS avg_price,
    RANK() OVER (PARTITION BY class ORDER BY AVG(price) DESC) AS price_rank,
    ROUND(AVG(price) - AVG(AVG(price)) OVER (PARTITION BY class), 0) AS diff_from_class_avg
FROM full_data
GROUP BY airline, class
ORDER BY class, price_rank
```
**Что узнали:** Vistara лидирует по средней цене в бизнес-классе (55 477 ₽), Indigo — самый дешёвый в экономе (5 324 ₽). Разница между лидером и аутсайдером в экономе — 1 970 ₽, в бизнесе — 8 346 ₽.

#### Запрос 2: Кумулятивная выручка по дням до вылета
```sql
WITH daily_revenue AS (
    SELECT days_left, SUM(price) AS revenue
    FROM full_data
    GROUP BY days_left
)
SELECT 
    days_left, revenue,
    SUM(revenue) OVER (ORDER BY days_left) AS cumulative_revenue,
    ROUND(revenue * 100.0 / SUM(revenue) OVER (), 2) AS pct_of_total
FROM daily_revenue
ORDER BY days_left
```
**Что узнали:** за 1 день до вылета генерируется 0.66% выручки. Основной объём идёт за 5–15 дней. Кумулятивно к 10-му дню накоплено всего 2.37% выручки.

#### Запрос 3: Конверсия по направлениям — доля каждого класса
```sql
WITH route_stats AS (
    SELECT source_city, destination_city, class,
           COUNT(*) AS bookings, ROUND(AVG(price), 0) AS avg_price
    FROM full_data
    GROUP BY source_city, destination_city, class
),
route_totals AS (
    SELECT source_city, destination_city, SUM(bookings) AS total_bookings
    FROM route_stats
    GROUP BY source_city, destination_city
)
SELECT 
    rs.*, rt.total_bookings,
    ROUND(rs.bookings * 100.0 / rt.total_bookings, 1) AS class_share_pct,
    RANK() OVER (PARTITION BY rs.source_city, rs.destination_city ORDER BY rs.bookings DESC)
FROM route_stats rs
JOIN route_totals rt ON rs.source_city = rt.source_city AND rs.destination_city = rt.destination_city
```
**Что узнали:** на всех популярных направлениях эконом занимает 65–71%, бизнес — 29–35%. Это стабильное соотношение, которое можно использовать для прогнозирования загрузки.

#### Запрос 4: Скользящее среднее цены за 7 дней
```sql
WITH daily_avg AS (
    SELECT days_left, ROUND(AVG(price), 0) AS avg_price
    FROM full_data
    GROUP BY days_left
)
SELECT 
    days_left, avg_price,
    ROUND(AVG(avg_price) OVER (ORDER BY days_left 
          ROWS BETWEEN 3 PRECEDING AND 3 FOLLOWING), 0) AS smoothed_avg_7d,
    avg_price - LAG(avg_price, 1) OVER (ORDER BY days_left) AS day_over_day_change
FROM daily_avg
```
**Что узнали:** цена не падает линейно. Колебания составляют ±2 500 ₽ относительно тренда. Миф «чем ближе к вылету, тем дешевле» не подтверждается.

#### Запрос 5: Топ-3 авиакомпании по каждому направлению
```sql
WITH route_airline AS (
    SELECT 
        source_city || ' → ' || destination_city AS route,
        airline, COUNT(*) AS flights_count,
        ROUND(AVG(price), 0) AS avg_price,
        ROW_NUMBER() OVER (PARTITION BY source_city, destination_city ORDER BY COUNT(*) DESC) AS rank
    FROM full_data
    GROUP BY source_city, destination_city, airline
)
SELECT * FROM route_airline WHERE rank <= 3
```
**Что узнали:** Vistara — лидер по количеству рейсов на всех маршрутах. Air_India — стабильно второй. Остальные конкурируют за третье место.

### Часть 2. Статистический анализ

#### Описательные статистики по классам

| Метрика | Economy | Business |
|---------|---------|----------|
| Среднее | 6 572 ₽ | 52 540 ₽ |
| Медиана | 5 772 ₽ | 53 164 ₽ |
| Стандартное откл. | 3 744 ₽ | 12 969 ₽ |
| IQR | 3 573 ₽ | 15 211 ₽ |
| Коэф. вариации | 57% | 24.7% |

**Вывод:** эконом — высоковолатильный рынок (CV=57%), бизнес — предсказуемый (CV=24.7%). Это влияет на UX: в экономе нужен трекер цен, в бизнесе — стабильность и сервис.

#### Процентили цен

| Процентиль | Цена |
|-----------|------|
| P1 | 1 776 ₽ |
| P10 | 3 389 ₽ |
| P25 | 4 783 ₽ |
| P50 (медиана) | 7 425 ₽ |
| P75 | 42 521 ₽ |
| P90 | 57 920 ₽ |
| P99 | 76 736 ₽ |

**Вывод:** резкий скачок между P75 и P50 (в 5.7 раз) — это граница между экономом и бизнесом. P90 = 57 920 ₽ — порог премиум-сегмента.

#### Правило Парето

Топ-10% самых дорогих билетов (дороже 57 920 ₽) приносят **32% всей выручки**.

#### 95% доверительный интервал

Средняя цена билета: **20 890 ₽**  
95% ДИ: **[20 808; 20 971] ₽**  
Интервал узкий благодаря большому объёму выборки (300K+). Это значит, что средняя цена рынка оценена с высокой точностью.

#### Распределение дней до вылета

| Процентиль | Дней до вылета |
|-----------|---------------|
| P10 | 7 |
| P25 | 15 |
| P50 | 26 |
| P75 | 38 |
| P90 | 45 |

**Вывод:** медиана — 26 дней. Это «точка принятия решения». Лучшее время для трекера цен и уведомлений.

### Часть 4. RFM-сегментация

Пользователи разбиты на 5 сегментов:

| Сегмент | Доля | Характеристика | Действие |
|---------|------|---------------|----------|
| **VIP** | ~5% | Часто, дорого, недавно | Персональный менеджер, закрытые тарифы |
| **Лояльные** | ~13% | Регулярно, средний чек | Повышать чек через бизнес-класс |
| **Перспективные** | ~25% | Недавно, но мало | Push-уведомления, трекер цен |
| **Спящие** | ~25% | Давно, редко | Одна скидка для реактивации |
| **Ушедшие** | ~32% | Очень давно | Не тратить бюджет |

**VIP + Лояльные (18%) приносят основной доход** — это ключевая аудитория для удержания.

### Часть 5. Бизнес-выводы и рекомендации

#### 1. Два класса — две вселенные
Средний чек бизнес-класса в 8 раз выше эконома. Разброс цен в экономе (CV=57%) создаёт тревогу у пользователя.

**Рекомендация:** внедрить прогноз цены и трекер «Лучшее время для покупки».

#### 2. Топ-10% билетов приносят 32% выручки
Премиум-сегмент критичен для бизнеса.

**Рекомендация:** премиум-подписка с приоритетной поддержкой и возвратом в 1 клик.

#### 3. Два типа пользователей
Планировщики (26 дней) ищут выгоду. Last-minute (7 дней) — скорость. Одинаковый интерфейс бесит обоих.

**Рекомендация:** разный UX. Трекер цен для первых, кнопка «Вылет сегодня» для вторых.

#### 4. Цена не падает линейно перед вылетом
Миф «ближе к вылету — дешевле» не подтверждается.

**Рекомендация:** показывать исторический график цен с меткой «Оптимальное окно покупки — 25 дней».

#### 5. Vistara и Air_India — 75% рынка
Две авиакомпании контролируют ¾ выручки.

**Рекомендация:** эксклюзивное партнёрство с Vistara (приоритетная выдача в поиске).

#### 6. 69% выбирают прямые рейсы
Главный критерий после цены.

**Рекомендация:** фильтр «Без пересадок» включён по умолчанию.

#### 7. VIP и лояльные — основной доход
18% пользователей приносят большую часть выручки.

**Рекомендация:** персональные предложения для VIP, одна скидка для спящих, исключить ушедших из рассылок.
